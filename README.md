# IQIYI-parser
 parse IQIYI url, and download the videos.

# 爱奇艺解析器 (IQIYI-parser)

	这是个成品程序，可以直接使用与下载爱奇艺视频。

## 项目包含
	1) /dl/*.py : dl目录下的是下载器。(github: https://github.com/ZSAIm/not-bad-downloader)
	2) iqiyi-parse.py : 视频解析。 ( / 目录下的两个js文件是用于解析地址的。)
	3) merge.py: 视频合并。
	4) /build/*: 已经编译好的程序。
