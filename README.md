# 爱奇艺解析器 (IQIYI-parser)

	爱奇艺视频下载器。【支持导入会员Cookie下载会员视频】

## 使用说明

* 打开程序 main.exe，然后自己摸索。。。
* 目录下的"Cookie.txt"可以导入Cookie。（在程序运行后写入Cookie.txt，重启程序生效。）
