# 爱奇艺、哔哩哔哩、腾讯视频解析器 (IQIYI-BILIBILI-parser)

	爱奇艺、哔哩哔哩、（腾讯视频）解析下载器。【支持导入会员Cookie下载会员视频】

# 注意
## 本程序仅供学习交流，请勿用于非法用途

## 使用说明

* 打开程序 main.exe，然后自己摸索。。。
* 目录下的"cookies/*.txt"可以导入Cookie。（爱奇艺导入iqiyi.txt，哔哩哔哩导入bilibili.txt）
